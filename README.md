# Predictive Analytics Using Historical Data

## Project Overview
This project focuses on predictive analytics using historical retail sales data. The main objective is to analyze past sales trends and build a machine learning model that can forecast future sales values.

The project uses Python and machine learning techniques to preprocess data, train a predictive model, evaluate its performance, and visualize predictions.

---

## Objective
To build a predictive analytics model that forecasts future sales trends using historical retail sales data.

---

## Dataset
The dataset contains historical retail sales information including:
- Order Date
- Customer Details
- Region
- Product Information
- Postal Code
- Sales

Dataset used: Superstore Sales Dataset

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## Machine Learning Algorithm
### Linear Regression
Linear Regression was used to predict future sales based on historical features such as:
- Year
- Month
- Postal Code

---

## Project Workflow

### 1. Data Collection
Imported historical sales dataset from CSV format.

### 2. Data Preprocessing
- Checked missing values
- Converted date columns
- Extracted Year and Month from Order Date

### 3. Feature Selection
Selected important numerical features for prediction.

### 4. Model Training
Trained the Linear Regression model using training data.

### 5. Prediction
Predicted future sales values using test data.

### 6. Model Evaluation
Evaluated performance using:
- Mean Absolute Error (MAE)
- R² Score

### 7. Data Visualization
Visualized actual and predicted sales using Matplotlib.
